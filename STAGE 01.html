HTML Markup

< name > Contents< /name> (ELEMENT)
< name > is opening tag
< /name> is closing tag

What is the tag?

A command inserted in a document that specifies how the document, or a portion of the document, should be formatted. Tags are used by all format specifications that store documents as text files.

Anchor Tag
< a href = "reddit.com">derp< /a>
< a> - anchor tag
href - attribute name
"reddit.com" - value

Image Tag
< img src = "url" alt = "text">
< img> - Images tag (VOID TAG - do not require closing tag)

Line Breaks
< br> - break (VOID TAG) - inline
< p> - paragraph (NOT VOID TAG) - block
